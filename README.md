# GoFrame Template For SingleRepo

Quick Start: 
- https://goframe.org/pages/viewpage.action?pageId=1114399

使用命令：

cpctl config update [应用名] [配置名] [配置值]
cpctl config view [应用名]

## docker-compose 操作
cpctl start [应用名]
cpctl stop [应用名]
cpctl restart [应用名]
cpctl status [应用名]

## kubernetes 安装
cpctl install [应用名]
cpctl uninstall [应用名]
